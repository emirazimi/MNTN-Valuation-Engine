# MNTN Valuation Engine

Python-based equity valuation model for MNTN using:

- Discounted cash flow (DCF)
- Monte Carlo simulation
- Empirical-Bayes priors from comparable companies
- Macro-aware regime switching
- EV/Revenue and EV/EBITDA cross-checks
- Sensitivity analysis and visualization

## Tools
Python, NumPy, pandas, SciPy, Matplotlib, SALib

- `mntn_valuation.py`: main valuation model
